并发编程模型的两个关键问题

  • 线程之间如何通信
    • 共享内存
    • 消息传递
  • 线程之间如何同步
    通信是指线程之间以何种机制来交换信息。在命令式编程中，线程之间的通信机制有两种：共享内存和消息传递。

Java内存模型的抽象结构

JMM：Java内存模型。
两个步骤：

  • 线程A把本地内存A中更新过的共享变量刷新到主内存中去。
  • 线程B到主内存中去读取线程A之前已更新过的共享变量。

as-if-serial语义
